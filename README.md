# AI Sales Call Assistant

## 📌 Project Overview
The AI Sales Call Assistant is an intelligent web application designed to assist sales teams during customer interactions. The system analyzes sales conversations, identifies customer sentiment, extracts important discussion points, and provides AI-based response suggestions to improve sales performance and customer engagement.

This project demonstrates the integration of Artificial Intelligence concepts with web development using Python and Flask.

---

# 🚀 Features

- 🎙️ Analyze sales call conversations
- 🧠 AI-based response suggestions
- 😊 Sentiment analysis of customer interactions
- 📄 Automatic conversation summarization
- 📌 Key discussion point extraction
- 💻 User-friendly web interface
- ⚡ Lightweight and fast Flask application
